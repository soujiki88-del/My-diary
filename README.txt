更新方法
1. ZIPを解凍
2. SafariでGitHubの My-diary を開く
3. Add file → Upload files
4. index.html / manifest.webmanifest / sw.js / icon-192.png / icon-512.png をアップロード
5. Commit changes
6. 2〜5分待つ
7. Safariで公開URLを再読み込み
8. ホーム画面版Secret Gardenを完全終了して開き直す

README.txt はアップロード不要です。
